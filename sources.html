Data Sources

Last updated: March 9, 2026

Important: BlueLedger is an independent civic transparency project. The current dashboard bundle does not ship as a government product, and the live bundle contains a mix of direct public-source imports, derived officer records, and auxiliary research files. This page distinguishes those categories so the shipped bundle is not overstated as purely official data.

Live Dashboard Bundle

The active dashboard bundle loaded by index.html reads only two data files: data/officers.parquet and data/incidents.parquet. A generated machine-readable inventory is available at data/manifest.json, with a readable summary in DATA_INVENTORY.md.

data/officers.parquet (6,620 rows)

This file is a mixed officer watchlist dataset. The shipped artifact now tags each row by source family when that can be determined from the stable identifier:

Tagged source families in the current bundle: 4,992 structured agency-record identifiers, 1,000 CCRB_* rows, 596 POST_* rows, and 32 legacy watchlist rows that remain unverified in the shipped bundle.

Important: the dashboard uses the unique record ID for officer selection. Badge numbers were missing or placeholder values for most rows and have been normalized accordingly.

data/incidents.parquet (22,780 rows)

This file is a tagged incident bundle composed of two named source families in the current artifact:

22,111 records with FE_* identifiers from Fatal Encounters and 669 records with IN_IMPD_* identifiers from Indianapolis / IMPD open-data imports.

Named Public Source Families Referenced by the Current Bundle

Fatal Encounters

National incident records used for the majority of the currently shipped incident bundle.

https://fatalencounters.org

Indianapolis / IMPD Open Data

Indianapolis Metropolitan Police Department incident imports represented by the IN_IMPD_* identifier family.

https://data.indy.gov/datasets/indygov::impd-officer-involved-shootings/about

NYC Civilian Complaint Review Board (CCRB)

The shipped officer bundle contains 1,000 CCRB_* identifiers, which are tagged as belonging to the New York City CCRB record family.

https://www.nyc.gov/site/ccrb/policy/data-transparency-initiative.page

Planned / Reference Sources (Not in Live Bundle)

Note: The following sources are referenced in research materials or planned for future integration, but are not currently included in the shipped parquet files. They should not be cited as active data sources for the live dashboard.

Mapping Police Violence

National database of police killings. Planned for future incident enrichment.

https://mappingpoliceviolence.org

Indiana Gateway (IFIONLINE)

Indiana fiscal transparency portal. Planned for salary/payroll enrichment.

https://gateway.ifionline.org

NPI Registry

National Provider Identifier registry. Referenced for cross-entity research only.

https://npiregistry.cms.hhs.gov

Auxiliary Files Present In-Repo But Not Loaded by the Live Dashboard

Research and News Feed Files

alerts.json, all_states.json, and states/*.json remain in the repo as auxiliary research/news assets. They are not referenced by the active dashboard bundle currently loaded from index.html.

Data Disclaimer

BlueLedger is not an official government database. Some bundled records are derived or only partially sourced in the shipped artifact. Use the bundle for research and transparency only, and verify any critical fact through an authoritative original source.
