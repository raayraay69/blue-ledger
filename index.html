BlueLedger

Local dashboard load notice

Loading the live dashboard downloads the bundled officer and incident parquet files, the compiled app assets, and a DuckDB WebAssembly runtime used to query the data locally in your browser.

Approximate download About 7.5 MB of bundled app/data files, plus the DuckDB runtime fetched at load time.
What the dashboard does not request No account signup, no precise location, no camera, no microphone, and no contacts access in the shipped bundle.
Current live data bundle `data/officers.parquet` and `data/incidents.parquet`. Auxiliary JSON/news files stay in-repo but are not loaded by the live dashboard.

BlueLedger is an independent research project and is not a government app. Some records in the shipped officer bundle are derived or only partially sourced. Review the source notes before relying on any entry.
